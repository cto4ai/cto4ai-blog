## What Are Cursor Rules?

Cursor Rules are markdown files (typically `.cursorrules` or files in `.cursor/rules/`) that provide context and guidelines to Cursor's AI assistant. They can include:

- Project structure and conventions
- Coding standards and best practices
- Framework-specific patterns
- Common workflows and commands
- Domain-specific knowledge

## What Are Claude Skills?

Claude Skills, introduced by Anthropic, are a more structured approach to providing context to Claude. They offer:

- Formal skill definitions with metadata
- Structured knowledge organization
- Reusable components across projects
- Version control and sharing capabilities
