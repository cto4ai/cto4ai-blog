# Entity Examples

## Link These (first occurrence only)

### People
- Researchers, founders, executives in tech/AI
- Conference speakers, podcast hosts
- Authors of referenced work

### AI Models & Technologies
- Current frontier models and their variants
- AI platforms and coding tools (Cursor, Windsurf, etc.)
- Protocols and frameworks (MCP, Agent SDK, etc.)

### Companies & Products
- Companies a general tech reader likely hasn't heard of
- Niche products and startups
- Developer tools outside mainstream awareness

### Internal References
- Self-references to other blog posts
- Use relative paths: `/p/post-slug`

### Acronyms & Jargon
- Industry terms that benefit from explanation
- Benchmarks (SWE-bench, MMLU, etc.)

## Skip These

### Household Names
- Apple, Google, OpenAI, Anthropic, Microsoft, Amazon, Meta, Nvidia
- GitHub, Twitter/X, YouTube, Slack

### Ubiquitous Products
- macOS, Windows, Linux, iPhone, Android
- Common dev tools everyone uses

### Basic Technical Terms
- REST, API, JSON, HTTP
- Database, server, client
- File formats (.pdf, .docx, .json)
